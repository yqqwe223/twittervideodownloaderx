> ⚠️ **تنبيه هام**: صُممت هذه الأداة حصرياً لأغراض التعلم الشخصي، والبحث التقني، والاستخدام ضمن حدود معقولة. يُرجى الالتزام بـ [اتفاقية مطوري منصة X](https://developer.twitter.com/en/developer-terms/agreement) وقوانين حقوق النشر السارية في بلدك. يُحظر تماماً استخدامها لأغراض تنتهك حقوق الآخرين.

## ✨ الميزات الرئيسية

- 🔗 **تحليل الروابط**: دعم روابط منشورات X.com القياسية؛ الكشف التلقائي عن مقاطع الفيديو وصور GIF المتحركة
- 📥 **تصدير بتنسيقات متعددة**:
  - فيديو MP4 بالجودة الأصلية
  - استخراج الصوت → تنسيق MP3
  - مقطع فيديو → تحويل إلى صورة متحركة GIF
- 🌍 **واجهة متعددة اللغات**: دعم اللغة العربية والإنجليزية والصينية واليابانية والكورية وما مجموعه 16 لغة
- 📱 **توافق متعدد المنصات**: يعمل بسلاسة على متصفحات Chrome / Firefox / Safari / Edge؛ تجربة مُحسّنة للأجهزة المحمولة والألواح اللوحية
- 🔒 **الخصوصية أولاً**: لا حاجة لتسجيل حساب، لا يتم جمع بيانات شخصية؛ عملية التحليل مجهولة الهوية بالكامل
- ⚡ **معالجة سريعة**: يكتمل التحليل في متوسط أقل من 5 ثوانٍ؛ دعم للطلبات المتزامنة

## 🚀 البدء السريع

### الاستخدام عبر الإنترنت (موصى به)
1. انتقل إلى [https://twittervideodownloaderx.com/ar](https://twittervideodownloaderx.com/ar)
2. انسخ رابط المنشور المستهدف (مثال: `https://x.com/user/status/123456789`)
3. ألصق الرابط في حقل الإدخال → انقر على زر 「تحليل」
4. اختر التنسيق المطلوب → احفظ الملف باتباع إرشادات المتصفح

### النشر المحلي (للمطورين)
```bash
# استنساخ المستودع
git clone https://github.com/your-repo/x-video-parser.git

# تثبيت التبعيات
cd x-video-parser && npm install

# تهيئة متغيرات البيئة (اختياري)
cp .env.example .env

# تشغيل خادم التطوير
npm run dev
```

> 💡 ملاحظة: يستخدم هذا المشروع بنية قائمة على Node.js + Express. راجع وثائق النشر التفصيلية في `/docs/DEPLOY.md`

## 📚 دليل الاستخدام

### سير العمل الأساسي
```
1. الحصول على رابط المنشور
   └─ افتح المنشور المستهدف على X.com → انسخ عنوان URL من شريط عناوين المتصفح

2. إرسال طلب التحليل
   └─ ألصق الرابط في حقل إدخال الأداة → انقر على 「تنزيل الفيديو」

3. اختيار تنسيق الإخراج
   ├─ 🎬 التنزيل بصيغة MP4: الحفظ بالجودة الأصلية
   ├─ 🎵 التحويل إلى MP3: استخراج مسار الصوت
   └─ 🎞 التحويل إلى GIF: إنشاء رسوم متحركة قصيرة (موصى به: ≤15 ثانية)

4. حفظ الملف
   └─ سيتم فتح المورد في علامة تبويب جديدة → نقرة بزر الماوس الأيمن/القائمة → 「حفظ باسم」
```

### نصائح للاستخدام على الأجهزة المحمولة
- نظام iOS (متصفح Safari): زر المشاركة → 「الحفظ في الملفات」
- نظام Android (متصفح Chrome): اضغط مطولاً على الفيديو → 「تنزيل الفيديو」
- في حالة التشغيل التلقائي للفيديو: انقر على `⋮` في الزاوية العلوية اليمنى من المشغل → اختر 「تنزيل」

## ❓ الأسئلة الشائعة

**س: أين يتم حفظ الملفات التي تم تنزيلها؟**  
ج: يتم حفظ الملفات في مجلد التنزيلات المُعد في متصفحك. يمكنك التحقق من هذا المسار أو تعديله في إعدادات المتصفح.

**س: هل يمكنني تحليل الحسابات الخاصة أو المحتوى المقيد؟**  
ج: لا. تعمل هذه الأداة فقط مع المنشورات المُعدة كعامة، وتحترم إعدادات الوصول للمحتوى الأصلي.

**س: هل تنخفض جودة الصورة/الصوت بعد التحويل؟**  
ج: يحافظ تنسيق MP4 على الجودة الأصلية. يستخدم تنسيق MP3 ترميزاً قياسياً بمعدل 128 كيلوبت في الثانية. يُحسّن تنسيق GIF معدل الإطارات وفقاً للمدة لتحقيق التوازن بين حجم الملف وسلاسة التشغيل.

**س: هل يتم تخزين سجل التنزيلات أو ذاكرة التخزين المؤقت؟**  
ج: لا. يتم نقل جميع الموارد مباشرة إلى جهاز المستخدم عبر وكيل مؤقت؛ لا يخزن الخادم أي طلبات أو ملفات وسائط.

**س: ماذا أفعل إذا فشل التحليل؟**  
ج: يُرجى التحقق من: ① ما إذا كان الرابط يشير إلى منشور عام صالح ② ما إذا كانت اتصالك بالإنترنت مستقراً ③ جرب استخدام متصفح آخر. إذا استمرت المشكلة، لا تتردد في الإبلاغ عنها عبر فتح قضية (Issue).
