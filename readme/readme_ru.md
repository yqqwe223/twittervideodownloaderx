## 📋 Обзор проекта

Этот проект представляет собой веб-инструмент для анализа видео, предназначенный для безопасного извлечения видеоресурсов из публичных публикаций на X.com (ранее Twitter) с возможностью конвертации и загрузки в различных форматах. Не требует установки клиентского ПО или регистрации аккаунта — используйте напрямую через браузер.

> ⚠️ **Важное уведомление**: Данный инструмент предназначен исключительно для личного обучения, технических исследований и использования в разумных пределах. Строго запрещено использование в целях, нарушающих права третьих лиц.

## ✨ Основные функции

- 🔗 **Анализ ссылок**: Поддержка стандартных URL публикаций X.com; автоматическое распознавание видео и анимированных GIF
- 📥 **Экспорт в нескольких форматах**:
  - Видео MP4 с исходным качеством
  - Извлечение аудио → Формат MP3
  - Видеофрагмент → Конвертация в анимированный GIF
- 🌍 **Многоязычный интерфейс**: Поддержка русского, английского, китайского, японского, корейского и ещё 16 языков
- 📱 **Кроссплатформенная совместимость**: Полная поддержка Chrome / Firefox / Safari / Edge; оптимизированный опыт для мобильных устройств и планшетов
- 🔒 **Приоритет конфиденциальности**: Регистрация не требуется, личные данные не собираются; процесс анализа полностью анонимен
- ⚡ **Высокая скорость обработки**: Анализ завершается в среднем менее чем за 5 секунд; поддержка параллельных запросов

## 🚀 Быстрый старт

### Использование онлайн (рекомендуется)
1. Перейдите по ссылке [https://twittervideodownloaderx.com/ru](https://twittervideodownloaderx.com/ru)

## 📚 Руководство по использованию

### Базовый рабочий процесс
```
1. Получить ссылку на публикацию
   └─ Откройте целевую публикацию на X.com → Скопируйте URL из адресной строки браузера

2. Отправить запрос на анализ
   └─ Вставьте ссылку в поле ввода инструмента → Нажмите 「Скачать видео」

3. Выбрать формат вывода
   ├─ 🎬 Скачать как MP4: Сохранить с исходным качеством
   ├─ 🎵 Конвертировать в MP3: Извлечь аудиодорожку
   └─ 🎞 Конвертировать в GIF: Создать короткую анимацию (рекомендуется: ≤15 секунд)

4. Сохранить файл
   └─ Ресурс откроется в новой вкладке → Правый клик/меню → 「Сохранить как」
```

### Советы для использования на мобильных устройствах
- iOS Safari: Кнопка «Поделиться» → 「Сохранить в Файлы」
- Android Chrome: Долгое нажатие на видео → 「Скачать видео」
- Если видео воспроизводится автоматически: Нажмите `⋮` в правом верхнем углу плеера → Выберите 「Скачать」

## ❓ Часто задаваемые вопросы

**В: Где сохраняются загруженные файлы?**  
О: Файлы сохраняются в папку загрузок, настроенную в вашем браузере. Вы можете проверить или изменить этот путь в настройках браузера.

**В: Можно ли анализировать приватные аккаунты или ограниченный контент?**  
О: Нет. Инструмент работает только с публикациями, настроенными как публичные, и уважает настройки доступа к исходному контенту.

**В: Снижается ли качество изображения/звука после конвертации?**  
О: Формат MP4 сохраняет исходное качество. Формат MP3 использует стандартное кодирование 128 кбит/с. Формат GIF оптимизирует частоту кадров в зависимости от длительности для баланса между размером файла и плавностью.

**В: Сохраняется ли история загрузок или кэш?**  
О: Нет. Все ресурсы передаются непосредственно на устройство пользователя через временный прокси; сервер не хранит никакие запросы или медиафайлы.

**В: Что делать, если анализ не удался?**  
О: Пожалуйста, проверьте: ① Соответствует ли ссылка действительной публичной публикации ② Стабильно ли ваше интернет-соединение ③ Попробуйте использовать другой браузер. Если проблема сохраняется, не стесняйтесь сообщить об этом через Issue.
